✅ Себестоимость звонка: посчитали

Сервер Timeweb RTX 3060 12GB: 12 000 ₽/мес. Qwen 2.5 7B INT4 (4 GB VRAM) + TTS Kokoro (2.5 GB) + STT своя модель (2 GB) = 8.5 GB из 12 GB. 2-3 одновременных звонка, ~1 500 звонков/мес.

Расчёт: 12 000 ₽ / 1 500 звонков = 8 ₽/звонок. При 200 звонках/клиент = ~1 067 ₽/мес на клиента. При 100 клиентах = 635 ₽/клиент (экономия на масштабе).

Статус: Посчитано

✅ Сравнение: свой сервер vs API

Свой сервер: ~8 ₽/звонок. Yandex GPT API: ~8 ₽/звонок. OpenAI API: ~7.5 ₽/звонок. Свой сервер дороже на 5-10%, но данные в РФ и полный контроль.

Вывод: Начать с API для быстрого запуска, параллельно разворачивать свой сервер. При 100+ клиентах свой сервер выгоднее.
